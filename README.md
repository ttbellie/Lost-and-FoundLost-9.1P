# Lost and Found Map App

This Android app helps users create and view lost or found item adverts with location-based features. Users can add item details, select a location using Google Places Autocomplete, or use their current location through GPS. Saved items are stored locally and can be displayed on Google Maps as markers.

The app also includes a radius-based search feature, allowing users to show only lost or found items within a selected distance from their current location. This makes it easier for users to find nearby items and improve the chance of returning lost belongings to their owners.

## Key Features

- Create lost or found item adverts
- Save item name, description, phone number, date, and location
- Select location using Google Places Autocomplete
- Get current location using GPS
- Store latitude and longitude for each advert
- Display all saved items on Google Maps
- Show nearby items using radius-based search
- SQLite local database storage
